# Sprint Planning Dashboard

A modern Sprint Planning Dashboard built with:

* Next.js 15
* TypeScript
* TailwindCSS
* Google Sheets API
* Vercel

## Features

* Public Read-Only Dashboard
* Google Sheets as Single Source of Truth
* Search
* Stream Filters
* Status Filters
* Expand / Collapse Sprint Cards
* Responsive Design
* Auto Refresh

## Deployment

This project is designed to be deployed on Vercel.
